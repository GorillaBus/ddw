Class: ModelDrawer

ModelDrawer()

Draw Model class objects on a 2D context (html canvas) by joining each of its points with lines, filling the shape with color or gradients and also rendering other effects like inner shadow or 2D context filters.

drawInnerShadow(model, lightSource, ctx, shadowScaleopt, shadowSpreadopt, shadowDisplacementopt)

Paints a circular inner shadow over the Model that depends on a light source.
Parameters:
Name Type Attributes Default Description
model Model A Model object to render its inner shadow.
lightSource Model A Model object that represents the source of the light
ctx Object A 2D rendering context object from the html canvas.
shadowScale Model <optional>
 6 A relation between the size of the Model and its inner shadow.
shadowSpread Model <optional>
 0.92 The rational length of the gradient that decays from full shadow to none.
shadowDisplacement Model <optional>
 0.94 A ratio that express how far away from the Model's center its shadow will appear.
