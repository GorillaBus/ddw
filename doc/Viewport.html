Class: Viewport

Viewport(settings)

new Viewport(settings)

View file: https://github.com/GorillaBus/ddw/blob/master/src/class/Viewport.js.

An extension of the body class that works as a view-reference. It provides functionality to be moved around (translated, rotated, scaled). It can be attached to other body objects in a scene to display the attached body's view. Scaling this object will work as a camera zoom.
The viewport object constructs a model positioned in the center of the screen by default.
