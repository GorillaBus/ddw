An Interaction type that represents DDW's main broad-phase collision detection and resolving mechanism. As a SpatialInteraction, collision checks are only done between a current _Body_ and the ones contained by the nearest neighboar cells of a spatial partitioning grid.

collision(BodyA, BodyB)

Verifies the geometrical intersection between two Body objects, based in their bounding circles. In case of a possitive intersection, it will resolve it computing the results of an elastic collision.
Parameters:
Name Type Description
